== Description ==

Advanced Link Directory is, simple, compact and quick-setup link directory plugin for WordPress. 
<p><strong>The main features are:</strong></p>
<ul>
<li>Option to display more detailed information on each link.</li>
<li>Advanced Link Directory is now easy to translate into any language and has a language folder with more info.</li>
<li>The ability to store notes for each link in the Control Panel.</li>
<li>Spam protection (links can be added only from the administrator’s control panel)</li>
</ul>

== Installation ==

Installation
Uploading via WordPress Dashboard

1. Navigate to the ‘Add New’ in the plugins dashboard
2. Navigate to the ‘Upload’ area
3. Select advlinkdirectory.zip from your computer
4. Click ‘Install Now’
5. Activate the plugin in the Plugin dashboard
6. Use this shortcode to add your links to any page: [show_link]
